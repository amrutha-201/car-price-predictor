# car-price-predictor
Developed a web-based Car Price Prediction system using Machine Learning. Cleaned and processed data with Pandas and trained a regression model using Scikit-learn. Integrated the model with a Flask web application to provide real-time predictions based on user input.
## Overview
This project predicts the price of used cars using Machine Learning. 
Users can enter car details like company, model, year, fuel type, and kilometers driven to get a predicted price.
## Technologies Used
Python
Pandas
NumPy
Scikit-learn
Flask
HTML
CSS
## Features
- Data preprocessing and cleaning
- Machine learning model for price prediction
- Web interface for user input
- Real-time price prediction
## How to Run
1. Install required libraries
pip install -r requirements.txt
2. Run the application
python app.py
3. Open browser
http://127.0.0.1:5000
